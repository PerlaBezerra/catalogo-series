# Catálogo de Séries

Projeto de uma página web que consome uma API pública de séries e exibe os dados na tela.  
A aplicação foi desenvolvida com HTML, Tailwind CSS e JavaScript usando Fetch API.

---

## Funcionalidades

- Listagem de séries via API  
- Busca por nome da série  
- Filtro por gênero  
- Modal com detalhes da série  
- Layout responsivo  

---

## API utilizada

https://api.tvmaze.com/shows  

---

## Tecnologias

- HTML  
- Tailwind CSS  
- JavaScript  
- Fetch API  

---

## Decisões técnicas

- Uso da API TVMaze por ser gratuita e fácil de usar  
- Uso de fetch para requisições assíncronas  
- Renderização dinâmica dos cards no JavaScript  
- Uso de modal para mostrar detalhes sem mudar de página  
- Limite de 50 séries para melhorar desempenho  

---

## Limitações

- Não possui paginação  
- Algumas séries podem não ter imagem ou nota  

---

## Possíveis melhorias

- Paginação dos resultados  
- Ordenação por nota  
- Sistema de favoritos  

---

## Como executar

- Baixe ou clone o projeto  
- Abra o arquivo `index.html` no navegador
